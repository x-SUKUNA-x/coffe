# ☕ Quick Start: Replace Images with Coffee

## 🎯 Fastest Way to Get Coffee Images

### Option 1: Use the Image Generator (5 minutes) ✨

I've created a **Coffee Image Generator** tool for you!

1. **Open the generator:**
   - Double-click `generate-coffee-placeholders.html` in your project folder
   - OR open it in your browser

2. **Download all images:**
   - The page will automatically generate 6 coffee cup images
   - Click the "💾 Download" button under each image
   - Save them with the exact filenames shown

3. **Replace the old images:**
   - Go to `public/images/` folder
   - Delete or rename the old drink files:
     - `brown-drink.webp`
     - `red-drink.webp`
     - `blue-drink.webp`
     - `orange-drink.webp`
     - `white-drink.webp`
     - `black-drink.webp`
   - Add your newly downloaded coffee images

4. **Refresh your browser:**
   - Go to http://localhost:5173/
   - You'll see coffee cups instead of drink cans! ☕

## 💡 Tips

1. **Keep filenames exact** - The code looks for specific filenames
2. **Use WebP format** - Better compression, but JPG/PNG work too
3. **Transparent backgrounds** - Not required, but look better
4. **High resolution** - At least 800px width for best quality

## 🆘 Having Issues?

### Images not showing?
- Check the filename is exactly correct (case-sensitive)
- Clear your browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Make sure files are in `public/images/` folder

### Want different colors?
- Edit `src/constants/index.js` to change `color` property
- Colors: brown, red, blue, orange, white, black
